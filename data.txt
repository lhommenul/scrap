<p><a href="https://travis-ci.org/shelljs/shelljs" rel="nofollow"><img src="https://img.shields.io/travis/shelljs/shelljs/master.svg?style=flat-square&amp;label=unix" alt="Travis" /></a>

<a href="https://codecov.io/gh/shelljs/shelljs" rel="nofollow"><img src="https://img.shields.io/codecov/c/github/shelljs/shelljs/master.svg?style=flat-square&amp;label=coverage" alt="Codecov" /></a>
<a href="https://www.npmjs.com/package/shelljs" rel="nofollow"><img src="https://img.shields.io/npm/v/shelljs.svg?style=flat-square" alt="npm version" /></a>
<a href="https://www.npmjs.com/package/shelljs" rel="nofollow"><img src="https://img.shields.io/npm/dm/shelljs.svg?style=flat-square" alt="npm downloads" /></a></p>
<p>ShellJS is a portable <strong>(Windows/Linux/OS X)</strong> implementation of Unix shell
commands on top of the Node.js API. You can use it to eliminate your shell
script's dependency on Unix while still keeping its familiar and powerful
commands. You can also install it globally so you can run it from outside Node
projects - say goodbye to those gnarly Bash scripts!</p>
<p>ShellJS is proudly tested on every node release since <code>v4</code>!</p>
<p>The project is <a href="http://travis-ci.org/shelljs/shelljs" rel="nofollow">unit-tested</a> and battle-tested in projects like:</p>
<ul>
<li><a href="http://getfirebug.com/" rel="nofollow">Firebug</a> - Firefox's infamous debugger</li>
<li><a href="http://jshint.com" rel="nofollow">JSHint</a> &amp; <a href="http://eslint.org/" rel="nofollow">ESLint</a> - popular JavaScript linters</li>
<li><a href="http://zeptojs.com" rel="nofollow">Zepto</a> - jQuery-compatible JavaScript library for modern browsers</li>
<li><a href="http://yeoman.io/" rel="nofollow">Yeoman</a> - Web application stack and development tool</li>
<li><a href="http://deployd.com" rel="nofollow">Deployd.com</a> - Open source PaaS for quick API backend generation</li>
<li>And <a href="https://npmjs.org/browse/depended/shelljs" rel="nofollow">many more</a>.</li>
</ul>
<p>If you have feedback, suggestions, or need help, feel free to post in our <a href="https://github.com/shelljs/shelljs/issues" rel="nofollow">issue
tracker</a>.</p>
<p>Think ShellJS is cool? Check out some related projects in our <a href="https://github.com/shelljs/shelljs/wiki" rel="nofollow">Wiki
page</a>!</p>
<p>Upgrading from an older version? Check out our <a href="https://github.com/shelljs/shelljs/wiki/Breaking-Changes" rel="nofollow">breaking
changes</a> page to see
what changes to watch out for while upgrading.</p>

<p>If you just want cross platform UNIX commands, checkout our new project
<a href="https://github.com/shelljs/shx" rel="nofollow">shelljs/shx</a>, a utility to expose <code>shelljs</code> to
the command line.</p>
<p>For example:</p>
<pre><code>$ shx mkdir -p foo
$ shx touch foo/bar.txt
$ shx rm -rf foo
</code></pre><h2><a id="user-content-plugin-api" class="deep-link" href="#plugin-api" aria-hidden="true" rel="nofollow"><svg aria-hidden="true" class="deep-link-icon" height="16" version="1.1" viewbox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Plugin API</h2>
<p>ShellJS now supports third-party plugins! You can learn more about using plugins
and writing your own ShellJS commands in <a href="https://github.com/shelljs/shelljs/wiki/Using-ShellJS-Plugins" rel="nofollow">the
wiki</a>.</p>
